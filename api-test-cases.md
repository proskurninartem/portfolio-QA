## Модуль 1: GET /api/users — Список пользователей

---

### TC-001: Получение списка пользователей (позитивный)

| Поле | Значение |
|---|---|
| **ID** | TC-001 |
| **Приоритет** | High |
| **Предусловия** | API доступен |

**Шаги:**
1. Отправить `GET https://reqres.in/api/users?page=1`

**Ожидаемый результат:**
- Статус-код: `200 OK`
- Тело ответа содержит поля: `page`, `per_page`, `total`, `total_pages`, `data`
- `data` — массив объектов с полями: `id`, `email`, `first_name`, `last_name`, `avatar`
- `page` = 1

**Статус:** ✅ PASS

## Модуль 3: POST /api/users — Создание пользователя

---

### TC-009: Создание пользователя с валидными данными

| Поле | Значение |
|---|---|
| **ID** | TC-009 |
| **Приоритет** | High |

**Тело запроса:**
```json

  "name": "Artem",
  "job": "QA Engineer"
}
```
